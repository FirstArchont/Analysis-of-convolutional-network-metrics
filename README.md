# Анализ показателей свёрточных сетей

## Описание
- В этом проекте был проведён анализ показателей свёрточных сетей на датасетах MNIST и CIFAR-10
- Было проведено сравнение свёрточных сетей с полносвязными моделями
- Было создано множество графиков на Python + matplotlib
- Были созданы модели с регуляризацией и residual блоками
- Были созданы модели с разными размерями ядер свёртки
- Были созданы модели с разными глубинами CNN
- Были реализованы различные кастомные слои
- Были исследованы разные residual блоки
